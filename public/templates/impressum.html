• FAQ


    1. Wie kann ich ein Spiel spielen?

    Nachdem Sie sich eingeloggt haben, steht ihnen eine Auswahl an verfügbaren spielen zur Verfügung. Tippen Sie auf das Spiel, dass sie gerne spielen möchten. Tippen Sie nun auf 'Spielen', um das Spiel zu starten.


    2. Wie kann ich mein Profil bearbeiten / löschen?

    Auf der eigenen Profilseite wird ein 'Profil bearbeiten'-Button angeboten. Hier ist es sowohl möglich die personenbezogenen Daten, als auch das persönliche Passwort zu ändern. Ebenfalls ist hier eine unkomplizierte Löschung des Accounts möglich.


    3. Werden meine Daten an Dritte weitergeleitet?

    Nein! Alle persönlichen Daten werden sehr vertraulich auf Servern, deren Standort in Deutschland ist, behandelt.


    4. Wie kann ich Spiele Abbonieren / Kündigen?

    Auf der Spieleauswahlseite sind alle verfügbaren Spiele aufgelistet. Beim tippen auf ein Spiel öffnet sich die Spieleinfo Seite zu dem jeweiligen Spiel. Dort kann ein Spiel abboniert, sowie gekündigt werden.


    5. Wie kann ich Spiele bewerten?

    Wenn ein Spiel abboniert wurde, kann auf der Spieleinfo Seite unter Kommentare eine Bewertung, sowie ein Kommentar zu diesem Spiel abgegeben werden.


    6. Kann ich mein Spiel pausieren und zu einem späteren Zeitpunkt weiterspielen?

    Ja, dies ist möglich und geschieht automatisch.


    7. Wie kann ich eigene Spiele erstellen?

    Dies ist im Moment noch nicht möglich, da sich dies noch in der Planungsphase befindet. In einem späteren Update dieser Applikation wird dies möglich sein.


    8. Kann ich mit anderen Spielern kommunizieren?

    Dies ist im Moment noch nicht möglich, da sich dies noch in der Planungsphase befindet. In einem späteren Update dieser Applikation wird dies möglich sein.

    Wie kann ich dieses Programm auf mein Smartphone installieren?
